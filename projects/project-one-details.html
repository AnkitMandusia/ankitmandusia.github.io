Project Overview

Kepler is a web application designed to answer complex questions about corporate financial documents. The system uses a **multi-agent architecture** where different AI specialists collaborate to handle user requests. Instead of just searching for text, it can understand a user's intent, extract structured data from financial tables, and generate both natural language answers and interactive data visualizations on the fly.
