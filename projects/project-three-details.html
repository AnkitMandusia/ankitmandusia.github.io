Veracity Engine: AI-Powered Fact-Checker

A next-generation RAG pipeline that verifies claims against real-time web evidence, vets sources, and applies LLM-based evaluation with self-consistency to ensure trustworthy verdicts.

Project Overview

Users submit a claim via text or voice. The system performs real-time web retrieval using the Tavily API, vets results against a trusted domain list, and extracts clean text with requests + BeautifulSoup. The Gemini LLM is queried multiple times in parallel, and a majority vote determines the verdict. Each answer is then evaluated using keyword overlap and LLM-as-a-judge metrics for quality, grounding, consistency, and relevance, with results logged for transparency.
